# HorizonEngine
HorizonEngine is an open source game engine built from the ground up mainly to power our own game **Catania**.

It currently only has very basic functionality and a somewhat finished editor but it will be expanded to support DirectX 11 & 12, Vulkan, OpenGL and Metal. 
Supported platforms will be Windows, MacOS and Linux and maybe Console coming later on.

Thats it for now if you're interested have a look around :)
