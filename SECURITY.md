# Security Policy

## Supported Versions

This is the list of current version of the engine and their support status. Usually only the newest version is "supported" which means in this case only that version will get patches.

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| 0.1.x   | :x:                |

## Reporting a Vulnerability

If you ever encounter a vulnerability inside the Engine you can either join our discord linked in our website or use the issues tab on GitHub.
